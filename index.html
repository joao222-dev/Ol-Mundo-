Vamos criar um site para aprender a usar o github, e para isso vamos usar o github pages, que é uma ferramenta do github que permite criar um site estático a partir de um repositório. Para isso, basta criar um repositório com o nome "username.github.io", onde "username" é o seu nome de usuário no github. Depois disso, basta criar um arquivo index.html e colocar o código do seu site dentro dele. Depois disso, basta fazer um commit e um push para o repositório e o seu site estará disponível em "https://username.github.io".
